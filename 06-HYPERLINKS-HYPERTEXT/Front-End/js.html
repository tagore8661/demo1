What is JavaScript?

JavaScript is a high-level, interpreted programming language that enables interactive web pages. It's an essential part of web applications alongside HTML and CSS, allowing you to create dynamic content, control multimedia, animate images, and much more.

üéØ Key Features of JavaScript

  • Dynamic Typing: Variables can hold different types of values
  • Interpreted Language: No compilation step required
  • Event-Driven: Responds to user interactions
  • Prototype-Based: Object-oriented programming support
  • First-Class Functions: Functions are treated as values
  • Cross-Platform: Runs in browsers and servers (Node.js)

üèóÔ∏è JavaScript Basics

Variables and Data Types:

// Variables
let name = "John";
const age = 25;
var city = "New York";

// Data Types
let number = 42;           // Number
let text = "Hello";        // String
let isActive = true;       // Boolean
let items = [1, 2, 3];     // Array
let person = {             // Object
    name: "Alice",
    age: 30
};
let nothing = null;        // Null
let undefined_var;         // Undefined

Functions:

// Function Declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// Function Expression
const add = function(a, b) {
    return a + b;
};

// Arrow Function (ES6)
const multiply = (a, b) => a * b;

// Function Call
console.log(greet("World"));

üéÆ DOM Manipulation

JavaScript can interact with HTML elements through the Document Object Model (DOM):

Selecting Elements:

// Select by ID
const element = document.getElementById("myId");

// Select by Class
const elements = document.getElementsByClassName("myClass");

// Select by Tag
const paragraphs = document.getElementsByTagName("p");

// Modern Selectors
const element = document.querySelector("#myId");
const elements = document.querySelectorAll(".myClass");

Modifying Elements:

// Change content
element.textContent = "New text";
element.innerHTML = "Bold text";

// Change attributes
element.setAttribute("class", "newClass");
element.src = "newImage.jpg";

// Change styles
element.style.color = "red";
element.style.fontSize = "20px";

üé≠ Event Handling

JavaScript responds to user interactions through events:

Common Events:

  • click: Mouse click
  • mouseover: Mouse hover
  • keydown: Key press
  • load: Page load
  • submit: Form submission
  • change: Input value change

Event Listeners:

// Add event listener
button.addEventListener("click", function() {
    alert("Button clicked!");
});

// Arrow function syntax
button.addEventListener("click", () => {
    console.log("Button clicked!");
});

// Remove event listener
button.removeEventListener("click", myFunction);

üîÑ Control Structures

Conditional Statements:

// If statement
if (age >= 18) {
    console.log("Adult");
} else if (age >= 13) {
    console.log("Teenager");
} else {
    console.log("Child");
}

// Switch statement
switch (day) {
    case "Monday":
        console.log("Start of work week");
        break;
    case "Friday":
        console.log("TGIF!");
        break;
    default:
        console.log("Regular day");
}

// Ternary operator
const message = age >= 18 ? "Adult" : "Minor";

Loops:

// For loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// While loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// For...of loop (arrays)
for (const item of items) {
    console.log(item);
}

// For...in loop (objects)
for (const key in person) {
    console.log(key + ": " + person[key]);
}

üì¶ Arrays and Objects

Array Methods:

let fruits = ["apple", "banana", "orange"];

// Add elements
fruits.push("grape");        // Add to end
fruits.unshift("mango");     // Add to beginning

// Remove elements
fruits.pop();                // Remove from end
fruits.shift();              // Remove from beginning

// Array methods
fruits.forEach(fruit => console.log(fruit));
const upperFruits = fruits.map(fruit => fruit.toUpperCase());
const longFruits = fruits.filter(fruit => fruit.length > 5);

Object Operations:

let person = {
    name: "John",
    age: 30,
    city: "New York"
};

// Access properties
console.log(person.name);        // Dot notation
console.log(person["age"]);      // Bracket notation

// Add/modify properties
person.email = "john@email.com";
person.age = 31;

// Object methods
const keys = Object.keys(person);
const values = Object.values(person);
const entries = Object.entries(person);

üöÄ Modern JavaScript (ES6+)

Template Literals:

const name = "Alice";
const age = 25;
const message = `Hello, my name is ${name} and I'm ${age} years old.`;

Destructuring:

// Array destructuring
const [first, second] = [1, 2, 3];

// Object destructuring
const {name, age} = person;

Spread Operator:

// Array spread
const newArray = [...oldArray, 4, 5];

// Object spread
const newPerson = {...person, email: "new@email.com"};

Promises and Async/Await:

// Promise
fetch("https://api.example.com/data")
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

// Async/Await
async function fetchData() {
    try {
        const response = await fetch("https://api.example.com/data");
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}

üìñ JavaScript Best Practices

  1. Use strict mode: "use strict" at the top of files
  2. Declare variables properly: Use let/const instead of var
  3. Use meaningful names: Clear variable and function names
  4. Handle errors: Use try-catch blocks
  5. Avoid global variables: Use modules and proper scoping
  6. Comment your code: Explain complex logic
  7. Use consistent formatting: Follow style guides
  8. Optimize performance: Avoid unnecessary DOM manipulation
