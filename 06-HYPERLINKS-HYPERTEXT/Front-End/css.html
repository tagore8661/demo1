üé® CSS (Cascading Style Sheets)

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. CSS controls the layout, colors, fonts, and overall visual appearance of web pages.

üéØ Key Features of CSS

  • Separation of Concerns: Separates content from presentation
  • Responsive Design: Create layouts that work on all devices
  • Powerful Selectors: Target specific elements precisely
  • Animations: Create smooth transitions and animations
  • Flexible Layouts: Grid, Flexbox, and positioning systems
  • Maintainable: Easy to update styles across entire websites

üèóÔ∏è CSS Syntax

CSS consists of rules with selectors and declarations:

selector {
    property: value;
    property: value;
}

/* Example */
h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

üìö CSS Selectors

Basic Selectors:

  • Element Selector: h1, p, div
  • Class Selector: .classname
  • ID Selector: #idname
  • Universal Selector: *
  • Attribute Selector: [attribute="value"]

Combinator Selectors:

  • Descendant: div p (p inside div)
  • Child: div > p (direct child)
  • Adjacent Sibling: h1 + p (p immediately after h1)
  • General Sibling: h1 ~ p (p siblings after h1)

Pseudo-classes:

  • :hover - Mouse hover state
  • :focus - Element has focus
  • :active - Element is being activated
  • :first-child - First child element
  • :nth-child(n) - Nth child element

üé® CSS Properties

Text and Font Properties:

  • color: Text color
  • font-family: Font typeface
  • font-size: Text size
  • font-weight: Text thickness (bold, normal)
  • text-align: Text alignment
  • line-height: Space between lines
  • text-decoration: Underline, strikethrough, etc.

Box Model Properties:

  • width, height: Element dimensions
  • margin: Space outside the element
  • padding: Space inside the element
  • border: Element border
  • box-sizing: How box dimensions are calculated

Background Properties:

  • background-color: Background color
  • background-image: Background image
  • background-size: Background image size
  • background-position: Background image position
  • background-repeat: Background image repetition

üì± CSS Layout Systems

Flexbox:

  • display: flex; - Creates flex container
  • flex-direction: Main axis direction
  • justify-content: Main axis alignment
  • align-items: Cross axis alignment
  • flex-wrap: Wrapping behavior
  • flex: Flex item properties

CSS Grid:

  • display: grid; - Creates grid container
  • grid-template-columns: Column definitions
  • grid-template-rows: Row definitions
  • grid-gap: Space between grid items
  • grid-area: Item placement

Positioning:

  • position: static; - Default positioning
  • position: relative; - Relative to normal position
  • position: absolute; - Relative to positioned ancestor
  • position: fixed; - Relative to viewport
  • position: sticky; - Sticky positioning

üé≠ CSS Animations and Transitions

Transitions:

  • transition-property: Which properties to animate
  • transition-duration: Animation duration
  • transition-timing-function: Animation easing
  • transition-delay: Animation delay

Animations:

  • @keyframes: Define animation steps
  • animation-name: Animation name
  • animation-duration: Animation duration
  • animation-iteration-count: Repeat count
  • animation-direction: Animation direction

üì± Responsive Design

Media Queries:

/* Mobile First Approach */
@media (min-width: 768px) {
    /* Tablet styles */
}

@media (min-width: 1024px) {
    /* Desktop styles */
}

Responsive Units:

  • %: Percentage of parent element
  • em: Relative to font size
  • rem: Relative to root font size
  • vw, vh: Viewport width/height
  • vmin, vmax: Viewport minimum/maximum

üîß CSS Methodologies

BEM (Block Element Modifier):

  • Block: .header
  • Element: .header__logo
  • Modifier: .header--fixed

CSS Preprocessors:

  • Sass/SCSS: Variables, nesting, mixins
  • Less: Dynamic stylesheet language
  • Stylus: Expressive CSS preprocessor

üöÄ Modern CSS Features

  • CSS Custom Properties (Variables): --main-color: blue;
  • CSS Grid: Advanced layout system
  • Flexbox: Flexible box layout
  • CSS Shapes: Text wrapping around shapes
  • CSS Filters: Visual effects
  • CSS Transforms: 2D and 3D transformations

üìñ CSS Best Practices

  1. Use external stylesheets: Separate CSS from HTML
  2. Organize your CSS: Group related styles together
  3. Use meaningful class names: Descriptive and semantic
  4. Avoid !important: Use specific selectors instead
  5. Mobile-first approach: Design for mobile, then desktop
  6. Optimize for performance: Minimize and compress CSS
  7. Use CSS reset/normalize: Consistent cross-browser styling
