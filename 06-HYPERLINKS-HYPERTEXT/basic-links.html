Basic HTML Links

What are Hyperlinks?

Hyperlinks (or simply "links") are connections between web pages or resources. They are created using the anchor tag <a> with the href attribute.

Basic Link Syntax

<a href="URL">Link Text</a>

Link Attributes

target Attribute

  • Open Google in New Tab (target="_blank")
  • Open GitHub in Same Tab (target="_self" - default)
  • Open Stack Overflow in Parent Frame (target="_parent")
  • Open W3Schools in Full Window (target="_top")

title Attribute

  • Google (hover to see tooltip)
  • GitHub (hover to see tooltip)
  • HTML Tutorial (hover to see tooltip)

rel Attribute

  • External Link (nofollow)
  • Secure External Link
  • Sponsored Link

Relative vs Absolute URLs

Absolute URLs (Full Path)

  • https://www.google.com
  • https://www.github.com/explore

Relative URLs (Relative to Current Page)

  • web-dev.html (same folder)
  • Front-End/html.html (subfolder)
  • ../README.md (parent folder)
  • ./basic-links.html (current folder)

Link States and Styling

Links have different states that can be styled:

  • :link - Unvisited link (default blue)
  • :visited - Visited link (default purple)
  • :hover - Link when mouse hovers over it
  • :active - Link when being clicked
  • :focus - Link when focused (keyboard navigation)

Best Practices for Links

  1. Use descriptive link text: Avoid "click here" or "read more"
  2. Make links visually distinct: Use color, underlines, or other styling
  3. Use target="_blank" sparingly: Only when necessary
  4. Add rel="noopener" with target="_blank": For security
  5. Provide context: Let users know where the link goes
  6. Test all links: Ensure they work and go to the right place
  7. Use HTTPS when possible: For secure connections
  8. Consider accessibility: Links should be keyboard navigable
