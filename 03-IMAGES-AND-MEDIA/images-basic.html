Image Attributes

Required Attributes:

  • src - Specifies the path to the image
  • alt - Provides alternative text for accessibility

Optional Attributes:

  • width - Specifies the width of the image
  • height - Specifies the height of the image
  • title - Provides additional information (tooltip)
  • loading - Controls when the image loads (lazy, eager)

Alt Text Guidelines

Good Alt Text Examples:

Alt text: "Red sports car parked in front of modern building"

Decorative image with empty alt text (alt="")

Alt Text Best Practices:

  • Be descriptive but concise
  • Don't start with "Image of..." or "Picture of..."
  • Include important text that appears in the image
  • Use empty alt="" for decorative images
  • Consider the context and purpose of the image

Common Image Paths

Relative Paths:

<img src="images/photo.jpg" alt="Description">
<img src="../images/photo.jpg" alt="Description">
<img src="./photo.jpg" alt="Description">

Absolute Paths:

<img src="/images/photo.jpg" alt="Description">
<img src="https://example.com/photo.jpg" alt="Description">
