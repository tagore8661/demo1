What are Nested Lists?

Nested lists are lists inside other lists, creating hierarchical structures. You can nest any type of list (ordered, unordered, or description) within any other type.

Important Rules for Nested Lists

  • Proper nesting: Child lists must be inside
  • elements
  • Indentation: Use proper indentation for readability
  • Closing tags: Make sure all lists are properly closed
  • Accessibility: Screen readers announce nesting levels
  • Don't over-nest: Too many levels can be confusing

Code Example

<ul>
    <li>Parent Item 1
        <ul>
            <li>Child Item 1</li>
            <li>Child Item 2
                <ul>
                    <li>Grandchild Item 1</li>
                    <li>Grandchild Item 2</li>
                </ul>
            </li>
        </ul>
    </li>
    <li>Parent Item 2</li>
</ul>

Best Practices

  • Use for hierarchical data: Perfect for outlines, menus, and organizational structures
  • Keep it logical: Make sure the hierarchy makes sense
  • Limit nesting depth: Generally avoid more than 3-4 levels
  • Use appropriate list types: Choose ordered vs unordered based on content
  • Test accessibility: Ensure screen readers can navigate the structure
