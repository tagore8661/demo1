HTML Description Lists (dl)

What are Description Lists?

Description lists are used to display terms and their definitions or descriptions. They consist of:

  • <dl> - Description list container
  • <dt> - Description term
  • <dd> - Description definition

FAQ Format

Frequently Asked Questions

What is the difference between HTML and XHTML?
XHTML is a stricter, more XML-based version of HTML that requires proper nesting, closing tags, and lowercase element names
Do I need to learn HTML before CSS?
Yes, HTML provides the structure that CSS styles, so understanding HTML is essential before learning CSS
Is HTML a programming language?
No, HTML is a markup language used to structure content, not a programming language that performs computations
What's the latest version of HTML?
HTML5 is the current standard, with ongoing development as a "living standard" by the WHATWG

When to Use Description Lists

  • Glossaries: Terms and their definitions
  • FAQs: Questions and answers
  • Specifications: Product features and details
  • Metadata: Name-value pairs
  • Contact information: Labels and contact details
  • Dictionaries: Words and meanings

Code Examples

Basic Description List:

<dl>
    <dt>Term 1</dt>
    <dd>Definition for term 1</dd>
    
    <dt>Term 2</dt>
    <dd>Definition for term 2</dd>
</dl>

Multiple Definitions:

<dl>
    <dt>Term</dt>
    <dd>First definition</dd>
    <dd>Second definition</dd>
</dl>

Best Practices

  • Use for term-definition pairs: Perfect for glossaries and specifications
  • Keep terms concise: Use brief, clear terms
  • Provide complete definitions: Make definitions comprehensive
  • Group related terms: Use separate lists for different topics
  • Consider accessibility: Screen readers handle description lists well
