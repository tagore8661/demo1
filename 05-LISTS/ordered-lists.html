HTML Ordered Lists (ol)

Basic Ordered List

An ordered list displays items in a numbered sequence:

  1. First item
  2. Second item
  3. Third item
  4. Fourth item

Recipe Example

How to Make Chocolate Chip Cookies

  1. Preheat oven to 375°F (190°C)
  2. Mix flour, baking soda, and salt in a bowl
  3. Cream butter and sugars in a separate bowl
  4. Beat in eggs and vanilla extract
  5. Gradually add flour mixture
  6. Stir in chocolate chips
  7. Drop spoonfuls of dough onto baking sheet
  8. Bake for 9-11 minutes until golden brown
  9. Cool on baking sheet for 2 minutes
  10. Transfer to wire rack to cool completely

List Attributes Summary

Attribute Description Values Example
type Numbering style 1, A, a, I, i <ol type="A">
start Starting number Any positive integer <ol start="5">
reversed Reverse order reversed (boolean) <ol reversed>
value Custom value for <li> Any integer <li value="10">

When to Use Ordered Lists

  • Sequential processes: Recipes, instructions, procedures
  • Ranked items: Top 10 lists, rankings, priorities
  • Step-by-step guides: Tutorials, how-to guides
  • Chronological events: Timelines, historical events
  • Hierarchical structures: Outlines, academic papers

Code Examples

Basic Ordered List:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Ordered List with Attributes:

<ol type="A" start="3" reversed>
    <li>Item C</li>
    <li>Item B</li>
    <li>Item A</li>
</ol>

Best Practices

  1. Use for sequential content: Only use ordered lists when order matters
  2. Keep items parallel: Use similar sentence structure for all items
  3. Be concise: Keep list items brief and to the point
  4. Consider accessibility: Screen readers announce the numbering
  5. Use appropriate numbering: Choose the type that fits your content
