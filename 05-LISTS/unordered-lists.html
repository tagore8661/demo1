List Attributes

Attribute Description Values Example
type Bullet style disc, circle, square, none <ul type="circle">

When to Use Unordered Lists

  • Non-sequential items: Shopping lists, feature lists
  • Navigation menus: Website navigation
  • Bullet points: Key points, benefits, features
  • Categories: Grouping related items
  • Options: Available choices or alternatives

Code Examples

Basic Unordered List:

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

Unordered List with Type:

<ul type="square">
    <li>Square bullet item</li>
    <li>Another square bullet</li>
</ul>

Best Practices

  • Use for non-sequential content: When order doesn't matter
  • Keep items parallel: Use similar sentence structure
  • Be concise: Keep list items brief and clear
  • Group related items: Use separate lists for different topics
  • Consider accessibility: Screen readers announce list structure
