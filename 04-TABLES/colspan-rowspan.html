Colspan and Rowspan Examples

What are Colspan and Rowspan?

  • colspan - Makes a cell span across multiple columns
  • rowspan - Makes a cell span across multiple rows

Important Rules for Colspan and Rowspan

  1. Count your columns: Make sure each row has the correct total number of columns
  2. Don't overlap: Cells with rowspan affect the rows below them
  3. Plan ahead: Draw your table structure before coding complex layouts
  4. Accessibility: Complex tables can be harder for screen readers to navigate

Common Mistakes to Avoid

  • Forgetting to account for spanned cells in subsequent rows
  • Creating uneven row structures
  • Using colspan/rowspan values that exceed available space
  • Not considering how the table will look on mobile devices

Code Examples

Colspan Example:

<tr>
    <th colspan="3">Header spanning 3 columns</th>
</tr>
<tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
    <td>Cell 3</td>
</tr>

Rowspan Example:

<tr>
    <td rowspan="2">Spans 2 rows</td>
    <td>Cell 1</td>
</tr>
<tr>
    <td>Cell 2</td>
</tr>
