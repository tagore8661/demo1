Basic HTML Tables

Table Structure

HTML tables are built using these elements:

  • <table> - Container for the entire table
  • <tr> - Table row
  • <td> - Table data cell
  • <th> - Table header cell

Table Elements Explained

Element Purpose Example
<table> Defines the table container <table>...</table>
<tr> Defines a table row <tr>...</tr>
<td> Defines a table data cell <td>Data</td>
<th> Defines a table header cell <th>Header</th>

Table Attributes

Attribute Element Description
border <table> Adds border to table (deprecated, use CSS instead)
width <table>, <td>, <th> Sets width (deprecated, use CSS instead)
height <table>, <td>, <th> Sets height (deprecated, use CSS instead)
align <table>, <td>, <th> Sets alignment (deprecated, use CSS instead)

Basic Table Code Example

<table border="1">
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
        <th>Header 3</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
        <td>Data 3</td>
    </tr>
    <tr>
        <td>Data 4</td>
        <td>Data 5</td>
        <td>Data 6</td>
    </tr>
</table>

Important Notes

  • Each <tr> should have the same number of <td> or <th> elements
  • Use <th> for headers, <td> for data
  • The border attribute is deprecated - use CSS for styling
  • Tables should be used for tabular data, not for layout
  • Always consider accessibility when creating tables
