Paragraphs and Line Breaks

Regular Paragraphs

This is a regular paragraph. HTML automatically adds space before and after paragraphs. Multiple spaces and line breaks in your HTML code are collapsed into single spaces when displayed.

This is another paragraph. Notice the space between this paragraph and the previous one.

Line Breaks

This line has a line break
right here. The br element creates a single line break.

You can use multiple line breaks

to create more space, but this is not recommended for styling purposes.

Preformatted Text

The pre element preserves both spaces and line breaks:

This is preformatted text.
    It preserves    multiple spaces
        and line breaks
            exactly as written.
    
    Perfect for:
    - Code examples
    - Poetry
    - ASCII art

Whitespace Handling

HTML collapses whitespace. This means that:

  • Multiple spaces become one space
  • Line breaks are ignored (except in pre elements)
  • Tabs are treated as spaces

For example, this text: has many spaces, but they are collapsed to single spaces.

Non-Breaking Space

Use &nbsp; to create spaces that won't break:

Name:     John Doe

Age:      25 years
